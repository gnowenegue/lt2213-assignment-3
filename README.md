# LT2213 Assignment 3 - Distributed Representations and Language Models

This repository contains the materials for Assignment 3 of LT2213 Computational Semantics.

## Setup

1. **Create a virtual environment:**
   ```bash
   python -m venv .venv
   ```

2. **Activate the virtual environment:**
     ```bash
     source .venv/bin/activate
     ```

3. **Install dependencies:**
   ```bash
   pip install -r packages.txt
   ```

4. **Install pre-commit hooks:**
   ```bash
   pre-commit install
   ```

## Contents

- `Word Embeddings and Language Models.ipynb`: Main assignment notebook.
- `data/`: Directory containing datasets.
- `packages.txt`: List of required Python packages.
